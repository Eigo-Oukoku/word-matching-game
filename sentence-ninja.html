🥷 忍びの心得 — Sentence Ninja

Read the Japanese sentence and build the English translation!
日本語の文を読んで、英語に並び替えよう！

🟢 Easy
First word placed. Tap remaining word chips in order.
最初の単語はすでに置いてあります。残りの単語を正しい順番にタップしよう。
🟡 Normal
All word chips given. Tap them all in order.
すべての単語が表示されます。正しい順番にすべてタップしよう。
🔴 Hard
Spell each word letter by letter. First word given.
各単語をアルファベット1文字ずつ入力しよう。最初の単語は表示されています。
🟣 Very Hard
Spell every word from scratch.
すべての単語を最初から自分でつづろう。
🌌 Ninja Sense Mode
No JP clue, no audio — only alphabetically sorted letters per word! Build the full sentence. Alternative valid answers are also accepted.
日本語ヒントも音声もなし！各単語のバラバラになったアルファベットだけを手がかりに、文全体を組み立てよう。別の正しい表現も正解として認められます。
⚡ Speedster — 60 sec!
Very Hard + clock. Score = letters typed correctly (partial progress counted at timeout).
Very Hardに60秒の制限時間が加わります。正しく入力した文字数がスコアになります（時間切れ時の途中入力もカウントされます）。
🥷 Ninja Type Blitz
See the Japanese, type the English. Score = letters typed correctly (partial progress counted at timeout).
日本語文を見ながら、英語をタイピングするモードです。正しく入力した文字数がスコアになります（時間切れ時の途中入力もカウントされます）。
🎯 Stages（ステージ）
Stage 1–4 or Master (all 48 sentences).
ステージ1〜4、またはマスター（全48文）から選べます。
⭐ XP per Character — モードごとの経験値
Ninja Sense & Type Blitz: 2.5 XP / character
Speedmaster ⚡: 2.0 XP / character (0.8× Ninja Sense)
Rank bonuses are added at the end of timed runs.
忍者感覚・Type Blitz: 1文字2.5XP ／ Speedmaster: 1文字2.0XP
タイムアタック系はランクボーナスXPが加算されます。
📊 Status Card — ステータスカード
Lv. — Earn XP to level up, up to Lv. 1000!
ACC — Accuracy: your correct-answer rate across all attempts.
Streak 🔥 — Consecutive correct answers. Longer streaks = bonus XP!
Lv. = 経験値を貯めてレベルアップ（最大Lv.1000）
ACC = 正答率（全解答に対する正解の割合）
Streak = 連続正解数。長いほどボーナスXP大！
📜 Shadow Clone Scroll — 分身の書
Save all ninja data (level, name, design) as an encrypted code. Restore any time — works across all Ninja games.
レベル・名前・キャラデザインを暗号化して保存・復元。シリーズ全ゲームで同じ忍者データを引き継げます。
🏷️ 通り名 / Alias
Earn a title by hitting high scores! Choose your favorite from the Status menu to display it on your card.
高いゲームスコアが出せると通り名が解放されます。好きな通り名をステータスメニューから選ぼう。
🎯 Analysis Scroll — 分析の巻物
Records words you answered wrong or slowly, and shows them more often in future sessions.
ACC = Accuracy (正答率)  ·  SRS = Review weight / 復習の重み
間違えた・時間がかかった単語を記録し、出題頻度を自動調整。Copy / CSVエクスポートにも対応。
